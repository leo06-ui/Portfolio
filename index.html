Welcome

This simple, navy blue themed website is built for a Year 9 intermediate level. It demonstrates semantic HTML, responsive layout, and accessible navigation.

Skills shown: headings, lists, images, links, CSS styling, hover effects.

About This Project

This page is a short project example suitable for Year 9 students learning web design. It uses a navy blue colour palette and clear structure so content is easy to read.

  • Semantic HTML structure
  • Responsive layout for phones and desktops
  • Simple, readable typography

Topics

HTML Basics

HTML is the structure of a webpage. Use headings, paragraphs, lists and links to organise content.

CSS Styling

CSS controls how the page looks. This site uses variables for the navy theme and simple layout rules.

Accessibility

Good websites are accessible: use clear headings, descriptive links and sufficient colour contrast.
